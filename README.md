# Nora Coessens - Wedding Photography Website

A clean, professional website for Nora Coessens' wedding photography business built with HTML, CSS, and JavaScript.

## 📁 Structure

```
├── index.html           # Landing page (entry point)
├── portfolio.html       # Portfolio & services page
├── css/
│   ├── landing.css      # Landing page styles
│   ├── portfolio.css    # Portfolio page styles
│   └── style.css        # Shared styles (legacy)
├── js/
│   ├── landing.js       # Landing page scripts
│   └── portfolio.js     # Portfolio page scripts
└── README.md
```

## 🚀 Getting Started

### Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. Edit content in the HTML files as needed

### GitHub Pages Deployment
1. Push code to a GitHub repository
2. Go to **Settings → Pages**
3. Select **Deploy from a branch** and choose `main`
4. Your site will be live at `https://yourusername.github.io/noraphotography/`

## 💡 Next Steps

### Add Your Content
- **Location**: Update your location in the hero or contact section
- **Add portfolio images** (`portfolio.html`): Replace placeholder boxes with actual wedding photos
- **Pricing**: Add engagement photo rates and wedding day packages
- **Add testimonials**: Create a new section with client quotes as you complete weddings
- **Update contact form**: Integrate with Formspree or Netlify Forms for email

### Enhance Marketing
- **SEO**: Update meta descriptions and add keywords for local search
- **Social links**: Add Instagram and Facebook buttons in the header
- **Analytics**: Add Google Analytics to track visitor behavior
- **Blog**: Add a blog section with wedding photography tips

### Professional Setup
- **Custom domain**: Point a GoDaddy/Namecheap domain to your GitHub Pages URL
- **Email collection**: Add Mailchimp signup for newsletters
- **Booking system**: Consider integrating Calendly or Acuity Scheduling
- **Photo gallery plugin**: Use Lightbox or Fancybox for image enlargement

## 🎨 Customization

### Colors
Edit the CSS files to change the color scheme:
- Accent color: `#c9a876` (gold/tan)
- Dark text: `#1a1a1a`
- Light background: `#f9f9f9`

### Fonts
Currently using 'Segoe UI' from system fonts for performance. To add custom fonts, add Google Fonts in the HTML `<head>`:
```html
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700&display=swap" rel="stylesheet">
```
